Import Library

Import your anime / manga list from MyAnimeList or AniList directly into your library.

  1. Go to myanimelist.net → Profile → Export
  2. Click Export Anime List and complete the captcha
  3. Extract the downloaded .gz file to get the .xml file
  4. Upload the XML file below

Manga list export works the same way — choose Export Manga List instead.

  1. Go to anilist.co → Settings → Apps
  2. Scroll to Export and click Download JSON
  3. Upload the .json file below

Both anime and manga lists are included in a single AniList export.
